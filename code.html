                            // reverse the string
                            // logic
                            function reverseString(userString){
                                let revString = [];
                            
                                // reverse string using for loop
                                for (let i = userString.length - 1; i >= 0; i--) {
                                    revString += userString[i];
                                    
                                }
                                return revString
                            }
                            
                            // display the reversed string
                            // View
                            function displayString(revString){
                            // write message to page
                            document.getElementById("message").innerHTML = `Your string reversed is: ${revString}`
                            // show alert box
                            document.getElementById("alert").classList.remove("invisible");
                            }
getValues()

The getValues function collects the user string data, inserts it into the function that reverses the string and then takes that and inputs it into the display function which is our view and controls how the data is displayed.

reverseString()

The reverseString function takes a parameter of userString. It takes the string finds the last letter (userString.length - 1), and loops though the string array until it finds the first letter which has an index of 0. Lastly it assigns that reversed string to the revString array and returns it.

displayString()

The displayString function takes a parameter of revString and displays the output of the reversed string.
